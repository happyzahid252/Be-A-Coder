SQL Basics with Example

    -- Create a table named 'employees'
    CREATE TABLE employees (
        id INT PRIMARY KEY AUTO_INCREMENT,        -- Unique identifier for each employee
        name VARCHAR(100) NOT NULL,               -- Employee's name
        age INT NOT NULL,                          -- Employee's age
        department VARCHAR(50),                    -- Department where the employee works
        hire_date DATE DEFAULT CURRENT_DATE       -- The date when the employee was hired
    );
    
    -- Insert some sample data into the 'employees' table
    INSERT INTO employees (name, age, department) VALUES
    ('Alice Smith', 30, 'Engineering'),
    ('Bob Johnson', 25, 'Marketing'),
    ('Charlie Brown', 35, 'Finance');
    
    -- Query to select all employees
    SELECT * FROM employees;
    
    -- Query to find employees in the Engineering department
    SELECT * FROM employees WHERE department = 'Engineering';
    
    -- Update an employee's information
    UPDATE employees SET age = 31 WHERE name = 'Alice Smith';
    
    -- Delete an employee from the table
    DELETE FROM employees WHERE name = 'Bob Johnson';

Explanation:

  • CREATE TABLE: This command creates a new table called 'employees' with defined columns and data types.
  • INSERT INTO: This command adds new records into the 'employees' table.
  • SELECT: This command retrieves data from the table; the first query gets all employees, and the second query filters for those in the Engineering department.
  • UPDATE: This command modifies existing records in the table. Here, it updates Alice's age.
  • DELETE: This command removes records from the table, used here to delete Bob Johnson's entry.

Oracle SQL Basics with Example

    -- Create a table named 'departments'
    CREATE TABLE departments (
        department_id NUMBER GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,  -- Unique identifier for each department
        department_name VARCHAR2(50) NOT NULL,                               -- Name of the department
        location VARCHAR2(100)                                               -- Location of the department
    );
    
    -- Insert sample data into the 'departments' table
    INSERT INTO departments (department_name, location) VALUES
    ('Sales', 'New York'),
    ('Marketing', 'Los Angeles'),
    ('Engineering', 'San Francisco');
    
    -- Query to select all departments
    SELECT * FROM departments;
    
    -- Query to find a department by name
    SELECT * FROM departments WHERE department_name = 'Sales';
    
    -- Update a department's location
    UPDATE departments SET location = 'Chicago' WHERE department_name = 'Marketing';
    
    -- Delete a department from the table
    DELETE FROM departments WHERE department_name = 'Sales';

Explanation:

  • CREATE TABLE: This command creates a new table called 'departments' with defined columns and data types.
  • INSERT INTO: This command adds new records into the 'departments' table.
  • SELECT: This command retrieves data from the table; the first query gets all departments, and the second query filters for a specific department.
  • UPDATE: This command modifies existing records in the table. Here, it updates the location of the Marketing department.
  • DELETE: This command removes records from the table, used here to delete the Sales department.

Python Basics with Example

    # Python program to demonstrate basic features
    
    # Function to greet a user
    def greet(name):
        return f"Hello, {name}!"
    
    # List of names
    names = ["Alice", "Bob", "Charlie"]
    
    # Loop through each name and greet them
    for name in names:
        print(greet(name))
    
    # Function to add two numbers
    def add(a, b):
        return a + b
    
    # Example usage
    result = add(5, 10)
    print(f"Result of addition: {result}")

Explanation:

  • Function Definition: The greet function takes a name as an argument and returns a greeting string.
  • List: A list of names is created to demonstrate looping through a collection.
  • Looping: The program loops through the names list and prints a greeting for each name.
  • Add Function: The add function takes two numbers as arguments and returns their sum.
  • Example Usage: The add function is called with the numbers 5 and 10, and the result is printed.
